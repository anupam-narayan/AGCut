Welcome to CNH AGCut

An intuitive UI tool that consolidates all internal automation utilities related to AG configuration.

Profile Checker

Supports the import and processing of profile JSON data to detect errors and uncover new CAN family entries.

Profile Summary

Import the profile summary sheet to ensure data remains current and up-to-date with the latest changes.

Compatibility Analyzer

Perform comparisons between JSON profiles and DBC files to assess reusability across equipment models.

Profile Diff

Perform a comparison between two profiles and highlight any differences for quick identification.

Spec Generator

Creates the Specs tables to insert in your documents with standardized formatting.
